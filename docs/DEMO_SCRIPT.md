## 📝 스크립트 (한국어)

### 씬 1: 훅 (0-10초)

**화면**: 스트레스 받는 크리에이터 (타임랩스)

**내레이션**:
```
영상 하나 만드는데 8시간?
매일 콘텐츠 올려야 하는데...
이제 AI가 대신 만들어 드립니다.
```

**자막**:
- 8시간 → 10분
- 수동 작업 → AI 자동화

### 씬 2: 솔루션 (10-20초)

```
OmniVibe Pro!
구글 시트에 전략만 적으면,
```

### 씬 3: AI 스크립트 생성 (20-30초)

**화면**: 스튜디오 화면 - 스크립트 생성 버튼 클릭

**내레이션**:
```
주제만 입력하면,
Claude AI가 완벽한 스크립트를 작성하고,
자동으로 장면을 나눠줍니다.
```

**데모**:
1. "신제품 런칭" 입력
2. "스크립트 생성" 클릭
3. 3초 만에 완성된 스크립트 표시

## 📹 촬영 가이드

### Equipment
- **Camera**: 4K (최소 1080p)
- **Mic**: Lavalier or Shotgun
- **Lighting**: 3-point lighting

### Screen Recording
- **Software**: OBS Studio / ScreenFlow
- **Resolution**: 1920x1080 @ 60fps
- **Cursor**: Highlight on click

### Voice Recording
- **Format**: WAV 48kHz
- **Environment**: Quiet room
- **Tone**: Energetic but professional
