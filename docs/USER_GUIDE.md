# OmniVibe Pro 사용자 가이드

> **5분 만에 첫 영상 만들기**

---

## 🚀 빠른 시작

### 1. 회원가입 (30초)

1. [omnivibepro.com](https://omnivibepro.com) 접속
2. "무료 시작" 클릭
3. 이메일/비밀번호 입력 또는 Google 로그인
4. 완료!

---

### 2. 첫 캠페인 만들기 (1분)

1. **대시보드** → "새 캠페인" 클릭
2. 캠페인 정보 입력:
   - 이름: "신제품 런칭 캠페인"
   - 플랫폼: YouTube
   - 톤: Professional
   - 스타일: Cinematic

3. "캠페인 생성" 클릭

---

### 3. AI 스크립트 생성 (2분)

1. 캠페인 상세 화면 → "콘텐츠 추가" 클릭
2. 주제 입력:
   ```
   AI 비디오 자동화 플랫폼 소개
   타겟: 콘텐츠 크리에이터
   길이: 60초
   ```

3. "스크립트 생성" 클릭
4. AI가 3초 만에 스크립트 작성!
5. 필요하면 수정 후 "저장"

---

### 4. 음성 생성 (1분)

#### 옵션 A: 기본 음성 사용

1. "오디오 생성" 클릭
2. 음성 선택: Korean Male (Default)
3. "생성 시작"
4. 약 30초 후 완성!

#### 옵션 B: 내 목소리 클로닝 (최초 1회, 5분)

1. "음성 클로닝" 클릭
2. 3-5분 녹음 (샘플 스크립트 제공)
3. "클로닝 시작"
4. 약 2분 후 내 목소리 완성!
5. 이후 모든 영상에서 사용 가능

---

### 5. 영상 렌더링 및 다운로드 (1분)

1. "영상 렌더링" 클릭
2. 배경 옵션 선택:
   - AI 생성 배경
   - 업로드한 이미지
   - 템플릿 선택

3. "렌더링 시작"
4. 약 2분 후 완성!
5. "다운로드" 또는 "플랫폼 배포"

---

## 📚 주요 기능 상세

### 스크립트 편집기

#### 블록 기반 편집

각 스크립트는 의미 단위로 "블록"으로 나뉩니다:

- **훅 (Hook)**: 시청자의 관심을 끄는 도입부
- **본문 (Body)**: 핵심 메시지 전달
- **CTA (Call-to-Action)**: 행동 유도

#### 블록 추가/삭제/순서 변경

- **추가**: "블록 추가" 버튼
- **삭제**: 블록 우측 상단 X
- **순서 변경**: 드래그 앤 드롭

---

### 음성 설정

#### 언어 지원

- 한국어 (Korean)
- 영어 (English)
- 일본어 (Japanese)

#### 음성 파라미터

- **속도**: 0.5x ~ 2.0x
- **피치**: -10 ~ +10
- **볼륨**: 0 ~ 100

---

### 플랫폼별 최적화

#### YouTube

- 해상도: 1920x1080 (16:9)
- 길이: 60-180초 권장
- 자막: 자동 생성

#### TikTok

- 해상도: 1080x1920 (9:16)
- 길이: 15-60초 권장
- 트렌드 음악 추가 가능

#### Instagram Reels

- 해상도: 1080x1920 (9:16)
- 길이: 15-90초 권장
- 정사각형 (1:1) 옵션

## 💡 팁 & 트릭

### 1. 고품질 스크립트 작성 팁

✅ **Do**:
- 명확한 주제 설정
- 타겟 관객 명시
- 구체적인 길이 지정

❌ **Don't**:
- 모호한 주제
- 너무 긴 스크립트 (180초 이상)

### 2. Voice Cloning 최적화

✅ **Do**:
- 조용한 환경에서 녹음
- 자연스러운 톤으로 읽기
- 3-5분 충분히 녹음

❌ **Don't**:
- 배경 소음 있는 환경
- 너무 빠르거나 느리게
- 1분 미만 짧게 녹음

### 3. 플랫폼별 최적화

**YouTube**:
- 긴 형식 (2-3분) 선호
- 정보성 콘텐츠
- SEO 최적화 (제목, 설명)

**TikTok**:
- 짧은 형식 (15-30초)
- 트렌드 활용
- 첫 3초가 중요

**Instagram**:
- 미적 감각 중시
- 정사각형 또는 세로 형식
- 해시태그 활용

## 🔧 문제 해결

### Q1: 오디오 생성이 실패했어요

**A**: 다음을 확인하세요:
- 스크립트에 특수문자나 이모지가 있나요? → 제거
- 텍스트가 너무 긴가요? (500자 이상) → 블록 나누기
- Quota가 남아있나요? → 플랜 확인

### Q2: 영상이 렌더링되지 않아요

**A**: 다음을 확인하세요:
- 오디오가 먼저 생성되었나요?
- 배경 이미지/영상이 업로드되었나요?
- 브라우저 캐시 삭제 후 재시도

### Q3: Voice Cloning이 부정확해요

**A**: 음성 녹음 품질을 높이세요:
- 더 조용한 환경
- 더 긴 녹음 시간 (5분 이상)
- 명확한 발음
